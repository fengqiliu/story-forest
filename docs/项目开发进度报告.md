### 4.1 积分商城功能

**功能点:**
- [x] 用户积分余额显示

- [x] 热门商品标记
- [x] 库存状态显示

**积分获取方式（规划）:**
- 每日签到 +10 积分
- 阅读故事 +20 积分/个
- 答题正确 +15 积分/题
- 连续打卡奖励

**可兑换商品:**
- 虚拟商品：故事会员、定制绘本
- 实物礼品：文具套装、徽章、台灯
- 会员服务：月卡、季卡、年卡
