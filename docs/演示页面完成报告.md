## 设计规范

### 色彩系统
```scss
主色调：森林绿 #2D6A4F
强调色：暖橙色 #FF8C42
辅助色：天空蓝 #4ECDC4
背景色：奶油白 #FFFBF0
```

### 字体系统
- 标题字：24px Semibold
- 正文字：16px Regular
- 辅助字：14px Regular

### 间距系统
- 页面边距：16px
- 卡片内边距：16px
- 元素间距：4px/8px/12px/16px/24px

### 圆角系统
- 小按钮：4px
- 卡片：8px/12px
- 大卡片：16px
- 圆形：9999px

## 技术栈

- **框架**: uni-app 3.x (Vue 3)
- **UI 库**: uView Plus

- **预处理器**: Sass
- **目标平台**: 微信小程序/H5/App

## 如何运行

### 1. 安装依赖
```bash
cd frontend
npm install
```

### 2. 开发模式
```bash
# 微信小程序
npm run dev:mp-weixin

# H5
npm run dev:h5

# App
npm run dev:app
```

### 3. 构建生产版本
```bash
npm run build:mp-weixin
npm run build:h5
npm run build:app
```
