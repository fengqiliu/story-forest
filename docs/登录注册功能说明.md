# 登录注册功能说明

**版本：** 1.0
**更新日期：** 2026-03-05

---

## 一、功能概述

### 1.1 用户流程

```
启动页 (splash)
    │
    ▼
检查登录状态
    │
    ├── 已登录 + 已设置年龄 → 首页
    ├── 已登录 + 未设置年龄 → 年龄选择页
    └── 未登录 → 登录页
```

### 1.2 页面列表

| 页面 | 路径 | 说明 |
|------|------|------|
| 启动页 | `/pages/splash/splash` | 品牌展示，自动检查登录状态 |
| 登录页 | `/pages/login/login` | 微信一键登录 / 手机号登录 |
| 注册页 | `/pages/register/register` | 手机号 + 验证码注册 |
| 年龄选择页 | `/pages/age-select/age-select` | 首次登录选择年龄段 |
| 用户协议 | `/pages/user-agreement/user-agreement` | 用户协议内容 |
| 隐私政策 | `/pages/privacy-policy/privacy-policy` | 隐私政策内容 |

## 二、功能详情

### 2.1 启动页 (Splash)

**功能:**
- 品牌 Logo 动画展示
- 自动检查本地登录状态
- 根据登录状态跳转到对应页面

**逻辑:**
```javascript
1. 检查 localStorage 中的 token
2. 有 token:
   - 检查 ageGroup 是否设置
   - 已设置 → 跳转首页
   - 未设置 → 跳转年龄选择页
3. 无 token → 跳转登录页
```

### 2.2 登录页 (Login)

**支持登录方式:**
1. **微信一键登录** (小程序)
   - 调用 `uni.login({ provider: 'weixin' })`
   - 获取 code 传递给后端
   - 后端换取 token 和用户信息

2. **手机号登录** (H5/App)
   - 输入手机号 + 验证码
   - 调用后端 API 验证

**登录成功后:**
- 保存 token 到 `localStorage.token`
- 保存用户信息到 `localStorage.userInfo`
- 检查是否设置年龄段，跳转对应页面

### 2.3 注册页 (Register)

**必填信息:**
- 手机号 (11 位)
- 验证码 (6 位)
- 同意用户协议和隐私政策

**可选信息:**
- 昵称
- 家长称呼

**注册流程:**
```
1. 输入手机号 → 点击「获取验证码」
2. 60 秒倒计时 → 接收短信验证码
3. 输入验证码 → 勾选同意协议
4. 点击「完成注册」
5. 保存登录状态 → 跳转年龄选择页
```

### 2.4 年龄选择页 (Age Select)

**年龄段选项:**
| 选项 | 图标 | 阶段说明 |
|------|------|----------|
| 3-4 岁 | 🍼 | 启蒙阶段 |
| 5-6 岁 | 🧸 | 幼儿阶段 |
| 7-8 岁 | 📚 | 小学低年级 |
| 9-10 岁 | 🎓 | 小学中年级 |

**功能:**
- 单选年龄段
- 触觉反馈 (小程序)
- 保存到本地用户信息
- 调用后端 API 更新用户年龄分组

## 三、API 接口

### 3.1 接口列表

| 接口 | 方法 | 说明 |
|------|------|------|
| `/auth/wechat-login` | POST | 微信登录 |
| `/auth/phone-login` | POST | 手机号登录 |
| `/auth/send-code` | POST | 发送验证码 |
| `/auth/register` | POST | 注册账号 |
| `/auth/logout` | POST | 退出登录 |
| `/user/current` | GET | 获取当前用户信息 |
| `/user/age-group` | PUT | 更新年龄分组 |
| `/user/profile` | PUT | 更新用户资料 |

### 3.2 请求格式

```javascript
// 请求示例
POST /auth/wechat-login
{
  "code": "wx_auth_code"
}

// 响应示例
{
  "code": 0,
  "message": "success",
  "data": {
    "token": "jwt_token_xxx",
    "userInfo": {
      "id": 1,
      "nickname": "微信用户",
      "avatar": "xxx",
      "ageGroup": null  // null 表示未设置
    }
  }
}
```

### 3.3 API 配置

修改 `frontend/src/api/auth.js` 中的 `BASE_URL`:

```javascript
const BASE_URL = 'https://api.example.com'  // 修改为实际 API 地址
```

## 四、本地存储

### 4.1 存储键值

| Key | 类型 | 说明 |
|-----|------|------|
| `token` | String | JWT 登录令牌 |
| `userInfo` | Object | 用户信息对象 |

### 4.2 userInfo 结构

```javascript
{
  "id": 1,              // 用户 ID
  "nickname": "xxx",    // 昵称
  "avatar": "xxx",      // 头像 URL
  "phone": "138xxxx",   // 手机号
  "ageGroup": "5-6",    // 年龄段 (3-4|5-6|7-8|9-10)
  "parentName": "xxx"   // 家长称呼
}
```

## 五、状态管理

### 5.1 userStore 使用

```javascript
import { userStore } from '@/store/user'

// 检查是否登录
userStore.isLogin()

// 获取用户信息
userStore.getUserInfo()

// 获取年龄分组
userStore.getAgeGroup()

// 设置用户信息
userStore.setUserInfo(userInfo, token)

// 更新用户信息
userStore.updateUserInfo({ nickname: '新昵称' })

// 退出登录
userStore.logout()
```
